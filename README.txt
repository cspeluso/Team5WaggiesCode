# Team5WaggiesCode
The code for team 5's accessible scale

This code requires 2 libraries to be downloaded:

-TFT_HX8357
  This link allows you to easily download this library: https://github.com/Bodmer/TFT_HX8357
-HX711
  This link offers instructions to install this library within Arduino: https://www.arduino.cc/reference/en/libraries/hx711-arduino-library/

If the load cell module is changed, the callibration factor (line 10) must be adjusted accordingly.
